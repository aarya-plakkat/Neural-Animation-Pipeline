# 🎯 Aarya VC - AI Avatar Setup Guide

## ✅ Step 3: Configure Your API Key

### Follow these steps EXACTLY:

1. **Open the application** at: http://localhost:8000

2. **Click the Settings (⚙️) icon** in the top right corner

3. **Scroll down to the "AI" section**

4. **Make sure "Gemini-1.5-pro" is selected** (should have a blue/highlighted background)

5. **Scroll down to "Gemini" section**

6. **Find the "API key" input field**

7. **Paste your Google Gemini API key** into that field
   - It looks like: `AIzaSy...` (starts with AIza)

8. **The key is saved automatically** in your browser's local storage

## ⚙️ Voice Settings

### To configure the voice:

1. **Go to Settings → Voice section**

2. **Select "Native" as Voice Type** (for browser speech synthesis)

3. **Choose your preferred voice** from the dropdown

4. **Adjust Rate and Pitch** sliders if needed

## 🐛 Troubleshooting

### If the avatar doesn't speak:

1. **Check voice type is set to "Native"**
2. **Make sure AI model is "Gemini-1.5-pro"**
3. **Verify API key is entered correctly**
4. **Check browser console (F12) for errors**

### If Gemini doesn't respond:

1. **Verify API key is correct** (copy-paste again)
2. **Check you have internet connection**
3. **Make sure "Gemini-1.5-pro" is selected**
4. **Look for error messages in console**

### If lip sync doesn't work:

1. **The visemes are generated automatically** from text
2. **Native speech triggers the lip animation**
3. **Check that the avatar loaded properly** (you should see the 3D model)
